# 🛡️ Mukeem Cyber Suite

An all-in-one terminal-based cybersecurity toolkit built with Python.

This toolkit combines multiple security and utility tools into a single powerful CLI application.

---

## 🚀 Features

### 🔍 1. Advanced Link Scanner
- HTTPS detection
- IP-based URL detection
- Suspicious keyword scanning
- Long URL detection
- Redirect symbol detection (@)
- Risk scoring system (Low / Medium / High)

---

### 📂 2. Smart File Organizer
- Automatically sorts files by type
- Creates folders (Images, PDFs, Python, Text, Others)
- Cleans messy directories
- Works on current or custom folder

---

### 🔐 3. Password Strength Checker
- Checks:
  - Length
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Provides strength score (0–5)
- Weak / Moderate / Strong rating

---

### 📜 4. Activity Log System
- Automatically logs:
  - URL scans
  - Folder organization
  - Password checks
- Saves data in `cyber_log.txt`

---

## 🖥️ How To Run

1. Make sure Python is installed.
2. Run:

```bash
python cyber_suite.py
```

---

## 📋 Menu Options

```
1 → Advanced Link Scanner
2 → File Organizer
3 → Password Strength Checker
4 → View Log File
exit → Quit
```
